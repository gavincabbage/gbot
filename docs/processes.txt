main gbot processes


SYSTEMS
=======

feed - pi camera feed (python/flask)
web - web interface (python/flask)
core - main controller (go)
sketch - arduino controllers (cpp)
redis - key/value store


BUILD
=====

0. Pre-requisites
    - Push latest code to GitHub

1. Run tests (TBD)

2. Build go binaries
    - core/main.go


DEPLOY
======

0. Pre-requisites
    - Flash arduinos manually

1. Pull code and binaries to pi

2. Install init script


RUN
===

0. Pre-requisites
    - Start redis
    - Flatten and load configuration into redis

1. Run init script

2. Startup script is called by init script
    - start core
    - start feed
    - start web
